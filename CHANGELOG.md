# 🥔 PotatoPS - Changelog

## Версия 1.0.0 (2026-03-27)

### ✨ Первый релиз

#### Модули:
- **📦 Установка ПО** - Менеджер программ на основе winget с каталогом 88+ приложений
- **🧹 Очистка системы** - Оптимизация, телеметрия, очистка temp, отключение служб
- **🔍 Диагностика** - SFC, DISM, CHKDSK, проверка RAM, сброс сети, логи ошибок
- **🤖 Удаление AI** - Copilot, Recall, AI пакеты, AI в Paint/Notepad
- **🗑️ Деблотер** - 20+ твиков реестра, интерфейса, проводника

#### Функции:
- Консольное меню с навигацией
- Категории программ (10 категорий)
- Пресеты для быстрой установки (4 пресета)
- Проверка обновлений winget
- Создание точек восстановления
- Просмотр системной информации
- Просмотр Wi-Fi паролей

#### Технические детали:
- PowerShell 5.1+
- Модульная архитектура
- Цветовой вывод информации
- Валидация прав администратора
- Проверка зависимостей
