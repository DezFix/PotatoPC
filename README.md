# 🥔 PotatoPS - Системный менеджер Windows

[![GitHub](https://img.shields.io/badge/GitHub-DezFix/PotatoPC-blue?style=for-the-badge&logo=github)](https://github.com/DezFix/PotatoPC)
[![PowerShell](https://img.shields.io/badge/PowerShell-5.1+-blue?style=for-the-badge&logo=powershell)](https://github.com/PowerShell/PowerShell)
[![License](https://img.shields.io/badge/license-MIT-blue?style=for-the-badge)](https://github.com/DezFix/PotatoPC/blob/main/LICENSE)

**PotatoPS** — универсальный модульный менеджер настройки Windows с консольным интерфейсом. Объединяет лучшие функции из проектов PotatoPC, RemoveWindowsAI и Win11Debloat.

![PotatoPS Menu](./Assets/preview.png)

---

## 🚀 Быстрый старт

### Способ 1: Запуск через PowerShell (рекомендуется)

```powershell
& ([scriptblock]::Create((irm "https://raw.githubusercontent.com/DezFix/PotatoPC/main/install.ps1")))
```

### Способ 2: Через короткую ссылку

```powershell
& ([scriptblock]::Create((irm "https://kutt.to/potatopc")))
```

### Способ 3: Локальный запуск

1. Скачайте репозиторий
2. Откройте PowerShell **от имени администратора**
3. Выполните команду:

```powershell
cd d:\CODE\PotatoPS_reborn\dist
.\launcher.ps1
```

### Способ 4: Через batch-файл

```cmd
.\Run.bat
```

## 📋 Требования

| Компонент | Требование |
|-----------|------------|
| **ОС** | Windows 10/11 (x64) |
| **PowerShell** | Версия 5.1 или выше |
| **Права** | Администратор (обязательно) |
| **winget** | Требуется для установки ПО |

## 📦 Модули

### 1. 📦 Установка ПО

Менеджер установки программ через **winget** с каталогом из 100+ приложений.

**Возможности:**
- Категории программ (Браузеры, Мультимедиа, Офис, Игры и др.)
- Пресеты для быстрой установки
- Проверка и установка обновлений
- Поиск программ по названию

**Пресеты:**
- 🏢 Офисный пакет
- 🎮 Геймерский набор
- 💻 Для разработчика
- 🏠 Базовый набор

---

### 2. 🧹 Очистка системы

Оптимизация и очистка Windows от мусора.

**Функции:**
- Очистка временных файлов
- Отключение телеметрии
- Настройка служб
- Оптимизация производительности
- Удаление встроенного ПО (Bloatware)
- Создание точки восстановления

---

### 3. 🔍 Диагностика

Проверка системных компонентов и исправление ошибок.

**Инструменты:**
- **SFC** — проверка системных файлов
- **DISM** — восстановление компонентов Windows
- **CHKDSK** — проверка диска
- **Memory Test** — проверка оперативной памяти
- **Сброс сети** — восстановление сетевых настроек
- **Просмотр ошибок** — анализ журнала событий
- **Инфо о системе** — полная информация о ПК
- **Wi-Fi пароли** — просмотр сохранённых паролей

---

### 4. 🤖 Удаление AI

Удаление и отключение AI-компонентов Windows.

**Возможности:**
- Отключение **Copilot**
- Отключение **Windows Recall**
- Удаление AI AppX пакетов
- Отключение AI-служб
- Отключение AI в **Paint** и **Notepad**
- Отключение **Click to Do**
- Установка классических приложений

> ⚠️ **Внимание:** Требуется PowerShell 5.1 (не PowerShell 7+)

---

### 5. 🗑️ Деблотер

Комплексное удаление встроенного мусора и твики реестра.

**Категории:**

#### Телеметрия и конфиденциальность
- Отключение телеметрии
- Отключение истории поиска
- Отключение Bing в поиске

#### Система
- Отключение быстрого запуска
- Отключение авто-шифрования BitLocker
- Отключение Storage Sense
- Настройка Windows Update
- Отключение OneDrive
- Удаление встроенных приложений
- Отключение Xbox Game Bar

#### Интерфейс
- Включение темной темы
- Отключение прозрачности
- Выравнивание панели задач влево
- Отключение виджетов
- Отключение чата
- Включение 'Завершить задачу'
- Скрытие Task View
- Отключение рекомендованного в Пуске

#### Проводник
- Показывать расширения файлов
- Показывать скрытые файлы
- Классическое контекстное меню

## ⚙️ Конфигурация

### apps.json

Файл `Config/apps.json` содержит каталог программ для установки.

**Структура:**
```json
{
  "Categories": {
    "🌐 Браузеры": [
      {"Name": "Google Chrome", "Id": "Google.Chrome", "Description": "..."}
    ]
  },
  "Presets": {
    "🏠 Базовый набор": ["Google.Chrome", "VideoLAN.VLC", "..."]
  }
}
```

Вы можете редактировать этот файл для добавления собственных программ.


## ⚠️ Предупреждения

> **Используйте на свой страх и риск!**

1. **Создайте точку восстановления** перед применением оптимизаций
2. **Не отключайте все службы** без понимания их назначения
3. **Удаление AI** может повлиять на работу некоторых функций Windows
4. **Деблотер** может удалить приложения, которые вы используете

## 🛠️ Решение проблем

### winget не найден

```powershell
# Установите winget
winget install --id Microsoft.WinGet
```

### Ошибка выполнения скрипта

```powershell
# Разрешите выполнение скриптов
Set-ExecutionPolicy Unrestricted -Scope Process -Force
```

### Модуль не загружается

Проверьте, что все файлы присутствуют в папке `Modules/`.
